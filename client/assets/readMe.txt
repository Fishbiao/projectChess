脚本基础规定：
================================================================================================================================================
                                                            命名规范
================================================================================================================================================
● 类的首字母必须大写 （如：Player、MgMsg）
● 数据类需加前缀Data （如：DataCommonParameter）
● 文件夹命名小写
● 同类型的类加前缀
● 类基础注释 类作用 类的作者

================================================================================================================================================
                                                           全局定义规范
================================================================================================================================================
● 单例模式类
● 本地数据管理类
● 基类
● 枚举
● 常用管理类、工具类

================================================================================================================================================
                                                    目录规范与分类 （Script:脚本目录）
================================================================================================================================================
● Components:    各类组件（不依赖当前项目）
    ● NetWork   ：网络通信相关
    ● ReCCC     : ccc组件重构 （比如按钮重构）
● Script    :    伪代码    
    ● Anim      :    动画文件 
    ● DoMain    ：   主要逻辑
    ● FrameWork :    主要是偏框架性的代码
        ● Cfg       :    主要存储配置、枚举等
        ● Manager   ：   主控类
        ● Msg       ：   弹框脚本
        ● Utils     ：   工具类 
    ● Msg     :  界面弹框类
    ● MgData  :  数据管理
        ● Data      :  游戏数据
        ● PlanData  :  策划表数据
    ● Scene   :  游戏场景类    
● Shared    :    公用类（主要是与服务器相关代码）

================================================================================================================================================
                                                            网络模块
================================================================================================================================================
● 游戏服务器 使用的是网易的pomelo开源框架
● 登录验证服务器（HTTP服务器）
